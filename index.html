My Journey

I started my development journey almost a year ago. Curiosity and a love for games and problem-solving pushed me forward.

I watched tutorials for two months, focusing on front-end HTML, CSS, and JavaScript. With the help of AI, I grew faster than I expected. Self-study worked well for me.

What began as experiments with HTML and CSS grew into an interest in full-stack development, user experience design, and modern web technologies.

At first, I avoided CSS. I felt lazy whenever I tried it. I realized I was more interested in problem-solving. I liked tackling questions such as, "How do I fix this error?" or "How do these components work?" The thrill was in solving logic puzzles, not calculating pixels to center a div.

This focus on logic and function shaped my approach to development. I aim to write clean, maintainable code. I build applications that work well and provide strong user experiences. Every project is a chance to learn and push further.
